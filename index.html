Bem vindos!

Bem-vindo ao site oficial da Maratona Inter-Universitária de Programação (MIUP) 2014, um concurso de programação destinado a alunos universitários que se realizará na FEUP no dia 25 de Outubro de 2014. A MIUP é um concurso por equipas (3 elementos) e as linguagens de programação permitidas para resolver os problemas propostos são o C, C++ e Java.

A MIUP serve como preparação dos alunos portugueses para o Southwestern European Regional Programming Contest (SWERC) 2014, que será realizado a 22 e 23 de Novembro de 2014 na Faculdade de Ciências da Universidade do Porto. Nalguns casos, a MIUP é também usada para selecionar os alunos que irão representar cada instituição no SWERC, mas esta é uma decisão interna de cada instituição universitária.

O SWERC é a prova regional do Sudoeste Europeu do ACM International Collegiate Programming Contest (ACM ICPC), servindo como prova de seleção para determinar quais as equipas que irão representar a nossa região na final mundial do ACM ICPC.
